## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd Campus-Evaluation-BE
```

Install dependencies:

```bash
npm install
```

## Running the Application

```bash
npm start
```

or

```bash
node server.js
```
